O jogo da Advinhação foi desenvolvido na linguagem JAVA seguindo um passo a passo de um livro que utiliza a linguagem C, "Introdução à programação em C" do autor Maurício Aniche. Para tornar mais simples no caso de alguém precisar estudar a linguagem JAVA, fiz o jogo em uma única página da IDE(EditPlus). Não há muitos arquivos.Embora não possua interface, por se tratar exclusivamente de primeiros passos no desenvolvimento de jogos, que tanto o código quanto a recomendação do livro, sejam úteis para os programadores.
